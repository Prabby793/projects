# My Projects
This is a repository of programs I wrote to learn Python. They are all in the programs/ directory.

1. Calculator.py - A simple calculator
2. Ski evaluator.py - A program that takes your ideal ski parameters as input and a list of available skis with parameters, and returns the closest one by calculating the Euclidean distance
3. Tic Tac Toe with Computer.py - A single playre tic tac toe
4. Tic Tac Toe without Computer.py - Two player tic tac toe
5. server.py - A server that listens to connections, exchanges messages and prints the messages it receives
